Mac下安装HomeBrew时因环境变量配置导致的报错

Mac下安装HomeBrew时因环境变量配置导致的报错

解决mac安装HomeBrew后报错-bash: brew: command not found
以及mac下安装HomeBrew 时出现警告：Warning: /usr/local/bin is not in your PATH.
方法一（单次解决）

终端输入export PATH=/usr/local/bin:$PATH 即可

方法二（永久解决）

输入 sudo vim .bash_profile

添加 export PATH=/usr/local/bin:$PATH

保存 export PATH=/usr/local/bin:$PATH

配置修改生效完成
